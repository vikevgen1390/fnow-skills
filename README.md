# fnow-skills — Personal Claude Code Skills Marketplace

Personal marketplace for Forecast NOW! implementation and training tools.

## Skills

| Skill | Description |
|---|---|
| `fnow-training-html` | Generates HTML training summaries from session transcripts: plan, Q&A, cases, interactive test, Opus judge |

## Installation

```bash
git clone https://github.com/vikevgen1390/fnow-skills ~/.claude/plugins/marketplaces/viktorov-market
```

## Adding new skills

Each skill lives in `plugins/<plugin-name>/skills/<skill-name>/SKILL.md`.
Follow the [superpowers/writing-skills](https://github.com/obra/superpowers) methodology.
